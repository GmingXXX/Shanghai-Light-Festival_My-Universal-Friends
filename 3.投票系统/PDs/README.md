# 🌌 宇宙朋友101 - 视频投票网站

一个零依赖、纯前端的视频投票系统，让你为心动的宇宙朋友投出宝贵的一票！

## ✨ 功能特色

- 🎯 **零配置使用**：双击 `index.html` 即可启动
- 🎲 **智能随机**：每次展示20个随机视频（不足20则全部展示）
- 🗳️ **灵活投票**：支持选择1-3个喜欢的视频
- 📊 **本地统计**：查看你的投票历史，按票数排序
- 📤 **数据导出**：支持导出为文本或Excel表格，方便统计分析
- 📱 **全端适配**：从手机到桌面的完美响应式体验
- 🌙 **苹果审美**：深色宇宙背景 + 霓虹渐变设计

## 🚀 快速开始

### 1. 使用方法
直接用浏览器（推荐 Chrome/Edge/Firefox）打开 `index.html` 即可开始使用！

### 2. 文件结构
```
宇宙朋友101投票/
├── index.html      # 主页面
├── style.css       # 样式文件  
├── script.js       # 逻辑文件
├── README.md       # 说明文档
└── video/          # 视频存放目录
    ├── 1.webm
    ├── 3.webm
    └── ... (更多视频)
```

### 3. 添加视频
将你的 `.webm` 视频文件放入 `video/` 文件夹，程序会自动识别并加载。

## 🎮 使用指南

### 投票流程
1. **浏览视频**：页面随机展示最多20个视频卡片
2. **选择喜爱**：点击"投TA一票"按钮（最多选3个）
3. **确认提交**：点击底部"确认提交我的投票"
4. **查看新朋友**：提交后自动刷新展示新的随机视频组合

### 统计查看
- 点击右上角"查看票数统计"按钮
- 查看所有投过票的视频及票数（降序排列）
- **导出功能**：支持导出为文本(.txt)或Excel表格(.csv)
- 可以重置投票记录（不可恢复）

## 🛠️ 技术特点

- **纯前端实现**：HTML + CSS + JavaScript，无需服务器
- **本地数据**：使用 localStorage 存储投票记录（仅本设备有效）
- **响应式设计**：移动端2列，平板3列，桌面4-5列自适应
- **无障碍支持**：完整的 ARIA 属性和键盘导航
- **性能优化**：视频懒加载、文档片段批量操作

## 📋 系统要求

- **浏览器**：Chrome 90+、Edge 90+、Firefox 90+、Safari 15+
- **视频格式**：WebM（推荐，兼容性最佳）
- **网络**：无需网络连接，完全本地运行

## 🎨 设计理念

采用深色宇宙主题配合霓虹渐变，营造科技感与未来感。界面设计遵循苹果公司的设计理念，注重简洁、直观和美观。

## 📤 分享使用

将整个项目文件夹打包成 ZIP 文件即可分享给他人。对方解压后可直接使用，但投票记录是独立的（基于各自设备的 localStorage）。

## ⚠️ 注意事项

- 投票数据仅保存在本地浏览器中
- 清除浏览器数据会导致投票记录丢失
- 不同设备/浏览器的投票记录互不影响
- 建议视频文件大小控制在合理范围内以确保流畅体验
